# Music Cue Sheet Generator

Converts Pro Tools "Export Session Info as Text" output into a clean
music cue sheet (CSV + PDF) for production company delivery.

## Requirements

- Python 3.10+
- reportlab (`pip3 install reportlab`)

## Usage

1. In Pro Tools, export session info as text (music tracks only or full session)
2. Update `input_file` path at the top of `cuesheet.py`
3. Run:# music-cue-sheet
